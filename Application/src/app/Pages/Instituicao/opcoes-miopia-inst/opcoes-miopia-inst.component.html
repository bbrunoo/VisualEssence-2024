MIOPIA

Jogos que ajudam a identificar a miopia nas crianças.

Letras

Este jogo divertido testa a miopia com letras que mudam de posição e tamanho! Em cada fase, o jogador tenta identificar letras que aparecem em diferentes ângulos, viradas para esquerda, direita, cima ou baixo, enquanto diminuem de tamanho. É uma aventura visual que desafia a visão e percepção de detalhes!

Objetos

Este jogo divertido testa a miopia com desenhos que ficam cada vez menores! Em cada fase, o jogador tenta identificar figuras que vão diminuindo de tamanho, desafiando a visão. É uma jornada visual divertida que ajuda a perceber como enxergamos detalhes de longe.
