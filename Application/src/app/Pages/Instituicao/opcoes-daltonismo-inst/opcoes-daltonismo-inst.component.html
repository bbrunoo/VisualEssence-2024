DALTONISMO

Jogos que ajudam a identificar o daltonismo nas crianças.

Números

Este jogo divertido ajuda a descobrir o daltonismo com desafios numéricos! Em cada fase, surgem números coloridos que o jogador precisa identificar corretamente. Cada nível traz novos números e cores, tornando a jornada emocionante enquanto revela como enxergamos e distinguimos diferentes tonalidades.

Animais

Este jogo divertido testa a miopia com desenhos que ficam cada vez menores! Em cada fase, o jogador tenta identificar figuras que vão diminuindo de tamanho, desafiando a visão. É uma jornada visual divertida que ajuda a perceber como enxergamos detalhes de longe.

Color blindness

Games that help identify color blindness in children.

Numbers

This fun game helps discover color blindness with challenges numeric! In each phase, colored numbers appear that the player need to identify correctly. Each level brings new numbers and colors, making the journey exciting while revealing how we see and distinguish different tones.

Animals

This fun game tests myopia with drawings that become increasingly smaller ones! In each phase, the player tries to identify figures that will shrinking in size, defying vision. It's a visual journey fun that helps us understand how we see details from afar.
