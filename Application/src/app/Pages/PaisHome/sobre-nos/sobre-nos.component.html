Code Fusion

Este projeto tem como finalidade proporcionar uma melhor qualidade de vida para as crianças que enfrentam no seu dia a dia as dificuldades causadas pela baixa visão.

No decorrer do website será apresentado de forma minuciosa e divertida uma possível ajuda, que auxiliará os responsáveis a obter uma direção por meio de jogos dinâmicos, para posteriormente analisar se estas crianças precisam recorrer a um especialista.

Devido a isso, os alunos do grupo Code Fusion do curso superior de Análise e Desenvolvimento de Sistemas, dedicam esse projeto a todas as crianças e responsáveis que encaram esse problema e necessitam de auxílio e apoio.
